Убираем надпись ***TAMPERED*** в бутлоадере HTC One M7

Требования
- S-Off
- Root-права
- Включенная отладка по USB

Приступаем:
1) На ПК запускаем командную строку
2) Переходим в папку с adb и fastboot (например, cd c:\android)
3) Убедимся, что телефон виден на ПК:
adb devices
Потом вводим следующие команды
adb shell

su

echo -ne '\x00' | dd of=/dev/block/mmcblk0p7 bs=1 seek=4265988 conv=notrunc

exit

exit

adb reboot bootloader

Для Вашего удобства каждая команда отделена друг от друга. Вводите осторожно, без опечаток!
