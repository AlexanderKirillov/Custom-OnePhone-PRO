Восстановление работы 4G
(HTC One E8 Dual Sim)
Требования:
- Компьютер на базе OS Windows c  установленными драйверами

1) Возвращаем стоковую прошивку.
    1.1) Качаем бэкап стока из шапки прошивок соответствующей темы вашего устройства и восстанавливаем его через кастомное рекавери. В данном случае, мы смотрим в шапку темы HTC One E8 Dual Sim-прошивки
   
2) Восстанавливаем стоковый рекавери.
    2.1) Находим и скачиваем файл стокового рекавери для Вашего устройства (найти можно в теме "Прошивки" вашего устройства на 4PDA)
    2.2) Закидываем данный файл в папку с файлами adb и fastboot.
    2.3) Последовательно из папки с  ADB и FASTBOOT даём команды:
--------------------------------------------
adb reboot-bootloader                               
fastboot flash recovery recovery.img           
fastboot erase cache                        
fastboot reboot
--------------------------------------------

3) Блокируем бутлоадер
    3.1) Перезагружаем аппарат в BootLoader
    3.2) Подключаемся к ББ, запускаем cmd.
    3.3) Входим в папку с ADB  и FASTBOOT и вводим
--------------------------------------------
fastboot oem lock
--------------------------------------------

4) После перезагрузки устройства делаем сброс на заводские настройки из меню телефона (Настройки->Резервное копирование и сброс-> Сбросить настройки телефона).
Галочки по удалению всех данных и очистку SD карты можно не ставить.
