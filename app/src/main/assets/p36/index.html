Как прошить ядро\рекавери без использования ПК
На данный момент существует программа для некоторых моделей HTC, которая позволяет прошить ядро\рекавери без участия ПК!
Интерфейс программы интуитивно понятный даже для самого начинающего пользователя.
Требуется Root доступ!

1) Качаем программу Recovery Tools Flasher и устанавливаем ее.
2) Кидаем файл ядра\рекавери во внутреннюю память (Формат файла- XXXXXX.img)
3) Заходим в программу и предоставляем ей Root доступ.
4) Выбираем требуемый пункт- "Flash kernel" или "Flash Recovery"
5) Следуем дальнейшим указаниям программы и прошиваем ядро\рекавери!
