Русификация стоковой прошивки

Для версий Android 4.1.х и ниже
1. Скачиваем с маркета программу Morelocale2
 2. Выбираем в ней язык системы
Для версий Android 4.2.х и выше
1. Скачиваем с маркета программу Morelocale2
2. Скачиваем на компьютер ADB, включаем в телефоне "отладку по USB" и подключаем к компьютеру
3.Скачанный архив необходимо распаковать в корень диска С так, чтобы файлы из него находились в C:\adb
4. Открываем командную строку Windows (CMD.exe) от имени администратора
5. Вводим туда 
Код
cd c:\adb\

6. Сначала проверяем, видит ли adb телефон
Код
adb devices
Должен быть отзыв как на скриншоте ниже:


7. Запускаем adb shell
Код
adb shell
8. Смотрим как называется пакет morelocale в системе
Код
pm list packages morelocale
Должен появиться ответ системы:


 9. Устанавливаем пакету дополнительные разрешения:
Код
pm grant jp.co.c_lis.ccl.morelocale android.permission.CHANGE_CONFIGURATION
Получаем ответ системы как на скриншоте ниже:


Если была допущена ошибка в команде, то система выложит помощь по pm.
10. В Morelocale2 выбираем язык системы
