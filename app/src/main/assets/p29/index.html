Ускорение поиска спутников GPS


Если вас не устраивает длительное время поиска спутников GPS на вашем устройстве, то вам нужна именно эта инструкция!

Для ускорения поиска спутников нам понадобится отредактировать файл  gps.conf, который можно найти по следующему пути: system/etc/gps.conf.

В этом файле, кроме всего прочего находится информация о NTP сервере, и нам нужно изменить конфигурацию, введя данные о ближайшем к нашему местоположению сервере. Если сервер выбран правильно,  любое устройство на поиск спутников будет тратить десятки секунд, а не несколько минут как раньше. 

Требования:

  •  Наличие Root прав

Инструкция:

1) Скачиваем любой файловый менеджер, работающий с Root правами (Например, Root Exploler)

2) Запускаем файловый менеджер

3) Переходим в каталог system/etc и находим файл gps.conf

4) Открываем редактор текста

5) Mеняем эту строку:

NTP_SERVER=north-america.pool.ntp.org

На строку (для России):

 NTP_SERVER=ru.pool.ntp.org

Если вы находитесь не в России, то строка для разных стран выглядит так:

Украина: NTP_SERVER=ua.pool.ntp.org

Беларусь: NTP_SERVER=by.pool.ntp.org

Германия: NTP_SERVER=de.pool.ntp.org

И так далее. Найти наименование ntp сервера для своей страны вы можете здесь

6) Сохраняем файл и перегружаем устройство!

 

Теперь вы можете проверить работу GPS. Поиск спутников теперь должен занимать менее одной минуты.
