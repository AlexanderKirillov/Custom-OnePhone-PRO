Установка кастомных прошивок с нуля

Сразу определитесь, будете ли вы разблокировывать бутлоадер. Это ведет к потере гарантии!

Если вы хотите установить пользовательскую версию ОС (Custom ROM) или кастомное ядро, вам нужно, чтобы ваш загрузчик был разблокирован. Говоря простым языком: разблокировка загрузчика позволяет устанавливать пользовательские прошивки телефона Android.

Плюсы разблокировки:
1) Установка кастомных ядер + нестираемые рут и рекавери в них
2) Неограниченая установка кастомных прошивок
3) Разгон ЦП

Минусы разблокировки:
1) Полная потеря официальной гарантии HTC (для РСТ телефонов)

Установка кастомов для разблокированного бутлоадера

1) Разблокировать бутлоадер (после этого произойдет сброс всех настроек)
2) Прошить любое кастомное ядро из шапки темы "Прошивки" Вашего устройства на 4PDA через fastboot (соблюдая одинаковые версии Android: ядра и прошивки. Если есть S-OFF- прошивать ядро не надо)
3) Скачать саму кастомную прошивку и закинуть ее на телефон
4) Перезагрузится и зайти в рекавери
5) В рекавери сделать:
На примере CWM:
Wipe User Data - не нужно делать при обновлении версии одной прошивки
Wipe Cache
Advanced => Wipe Dalvik Cache
Mounts and Storage => Format System
На примере TWRP:
Wipe-> Advanced Wipe-> выбираем все, кроме внутренней памяти (Internal Storage) и карты памяти (не нужно делать при обновлении версии одной прошивки)
6) В рекавери в главном меню нажать
На примере CWM:
Install zip from sdcard
На примере TWRP:
Install
Найти в папках вашу прошивку и подтвердить установку.
7) Перезагрузится и радоваться
